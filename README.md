# ExpressApi
This is a Powerful Express API set up using Sequelize
To Run Environment Port Be sure to open them up first

## How To Run? ##
```sh
npm start
```
or
```sh
yarn start
```

## To Use Environment Ports ##
For Command Prompt: 
```sh
set PORT=5000
```

For Power Shell: 
```sh
$env:PORT=5000
```

For Bash 
```sh
(Windows): export PORT=5000
```
